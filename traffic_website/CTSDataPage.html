Data

To draw conclusions, we obtained data from the city of Chicago's public API. The API, powered by Socrata and the affiliated Python library SodaPy, contains a vast amount of data on public administration within the city, including topics related to education, finance, and public safety. Since our project was concerned primarily with the relationship between car accidents and traffic cameras, we used datasets from the transportation category. The datasets we used are described in the table below along with a permanent link to the source, a brief description of the information, and the charts we formed using the dataset. We retrieved our data on March 19, 2019.
